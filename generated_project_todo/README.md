# TodoListApp

## Overview
TodoListApp is a simple web-based application that allows users to manage a personal to‑do list. Users can add new tasks, mark tasks as completed, and delete tasks they no longer need. The app runs entirely in the browser with no backend dependencies.

## Technology Stack
- **HTML** – Structure of the application.
- **CSS** – Styling and layout.
- **JavaScript** – Interactive functionality (add, complete, delete tasks).

## File Overview
- **`index.html`** – The main HTML file that defines the page layout, includes the stylesheet and JavaScript, and provides the UI elements for the to‑do list.
- **`styles.css`** – Contains all visual styles for the app, such as layout, colors, fonts, and responsive design.
- **`app.js`** – Implements the core logic: handling user input, creating list items, toggling completion status, and removing items.

## Features
- **Add Tasks** – Users can type a task into the input field and press *Enter* or click the add button to append it to the list.
- **Complete Tasks** – Clicking on a task toggles its completed state, visually indicated by a strike‑through style.
- **Delete Tasks** – Each task includes a delete button that removes the task from the list.

## Setup Instructions
1. Clone or download the repository to your local machine.
2. Open the `index.html` file in any modern web browser (e.g., Chrome, Firefox, Edge, Safari). You can simply double‑click the file or right‑click and choose “Open with…”.
3. No additional setup, server, or package installation is required—the app runs entirely client‑side.
